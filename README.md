# binary-search-basic
O exercício pede criar uma Árvore Binária de Busca (BST) a partir de números lidos de um arquivo e implementar a busca conforme o fluxograma: iniciar na raiz, comparar o valor desejado com o nó atual e seguir para a esquerda ou direita até encontrar o item ou concluir que ele não está na árvore.
